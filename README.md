## Description

This is a Blinky Board for stasis! Its very cool and when its powered it has a bunch of Leds "chasing" each other like after one turns on the one before it turns off and it goes through all the Led!

## Why I made It

I made it because this is one of my First Pcbs! I wanted something easy and people said to do this for the fundamentals. I wanted it to be useful so i made it chiikawa themed cause chiikawa makes me happy!
